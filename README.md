# 🚀 ConnectSmart AI

### AI-Powered Workplace Productivity Assistant

ConnectSmart AI is a modern AI-powered workplace productivity platform designed to help professionals streamline workflows, automate repetitive tasks, improve decision-making, and boost overall workplace efficiency.

By combining intelligent automation, productivity analytics, task management, meeting intelligence, and AI-powered communication tools, ConnectSmart AI empowers users to work smarter, save time, and reduce workplace burnout.

---

# 📖 Project Overview

In today's fast-paced digital workplace, professionals spend a significant amount of time managing emails, attending meetings, organizing tasks, and researching information.

ConnectSmart AI solves these challenges by providing an intelligent workplace assistant that automates routine tasks while delivering actionable insights and personalized productivity recommendations.

The platform combines the capabilities of workplace productivity tools with the power of Artificial Intelligence to create a centralized digital assistant that helps users stay organized, focused, and productive.

### Project Goals

* Increase workplace productivity
* Reduce time spent on repetitive tasks
* Improve communication efficiency
* Enhance task and workload management
* Reduce burnout through smart workload analysis
* Support data-driven workplace decisions

# ⚙️ Setup Instructions

## Prerequisites

Install:

* Node.js (v18+)
* npm
* Git

---

## Clone Repository

```bash
git clone https://github.com/yourusername/connectsmart-ai.git

cd connectsmart-ai
```

---

## Install Dependencies

```bash
npm install
```

---

## Configure Environment Variables

Create a `.env` file:

```env
OPENAI_API_KEY=your_openai_api_key
GEMINI_API_KEY=your_gemini_api_key
SUPABASE_URL=your_supabase_url
SUPABASE_ANON_KEY=your_supabase_anon_key
```

---

## Run Development Server

```bash
npm run dev
```

Application URL:

```text
http://localhost:3000
```

---

## Build for Production

```bash
npm run build
```
